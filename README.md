# Geometric Modeling
## Mesh Modeling
Unity project to generate meshes for multiple geometric forms.

## Description
This directory contains the source code and the build of a Unity project in which I developed scripts for generating the meshes of a plane, a cylinder, a sphere, a cone, and a truncated sphere, each with various configurable parameters. 
The build allows you to observe the correct functioning of the scripts by clicking buttons to generate the different meshes.
